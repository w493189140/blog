    去掉hash的#

    hashbang
    默认值： true
    只在 hash 模式下可用
    当 hashbang 值为 true 时，所有的路径都会被格式化为以 #! 开头。例如 router.go(‘/foo/bar’) 会把浏览器的 URL 设为 example.com/#!/foo/bar 。
